# SDS Global Dashboard

Safety Data Sheet (SDS) management system for Salesforce Health & Safety operations across 9 offices in the US and Canada.

## Project Overview
Centralized web-based dashboard for storing, accessing, and managing SDS documents with Google Cloud integration.

## Key Features
- Searchable document library with multi-office filtering
- Document upload system with OCR metadata extraction
- User role management (Admin, Editor, Viewer)
- Compliance tracking and expiration alerts
- Analytics dashboard for H&S management
- Google Workspace SSO integration

## Tech Stack
- **Frontend**: React.js or Vue.js with Material-UI
- **Backend**: Node.js (Express) or Python (FastAPI)
- **Database**: Google Cloud Firestore or Cloud SQL (PostgreSQL)
- **Storage**: Google Cloud Storage
- **Hosting**: Firebase Hosting or Google Cloud Run
- **Auth**: Google Cloud Identity Platform / Firebase Auth
